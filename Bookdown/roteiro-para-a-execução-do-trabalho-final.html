ESTUDOS ECOLÓGICOS

14 Roteiro para a execução do trabalho final

1- Este trabalho poderá ser feito de forma individual ou em dupla;

2- Este trabalho será dividido em duas etapas. A primeira etapa deverá ser baseada em uma análise do tipo Série Temporal (ST) e a segunda em uma Análise Espacial (AE). A primeira etapa irá valer 4,0 pontos e a segunda 6,0 pontos;

3- Em cada etapa poderá ser feita análises com tipos de dados diferentes, ou seja, com desfechos ou estudos distintos para cada tipo de análise;

4- Em ambas as etapas, o tema analisado deverá ser descrito sucintamente assim como as variáveis analisadas, sempre especificando a unidade espacial (ex: setor censitário, bairro, município, estado, etc.) e a temporal (ex: dia, semana, mês, ano, etc) que serão levadas em consideração. Caso os dados sejam os mesmos em ambas as etapas, tal descrição poderá ser feita apenas uma vez;

5- A primeira etapa poderá ser feita utilizando o modelo ARIMA e/ou GAM. Nesta etapa deverá constar os seguintes procedimentos para as análises: Ler o banco de dados pretendido, transformar em formato de ST, decompor a ST e testar a presença dessas componentes, ajustar os modelos e testar os pressupostos. Interpretar o modelo. Avaliar o processo de modelagem como um todo.

6- A segunda etapa poderá ser feita utilizando os tipos de dados em padrão pontual e/ou em área. Nesta etapa deverá constar um dos seguintes procedimentos para as análises:

6.1 – Padrão Pontual

  1. Ler o banco de dados pretendido
  2. Transformar em formato espacial
  3. Explorar o Processo de primeira ordem (Ex: Testar CSR, Função Kernel)
  4. Explorar o Processo de segunda ordem (Ex: Função K, F, L)
  5. Modelagem (Ex: GAM)

6.2 – Dados de Área

  1. Ler o arquivo vetorial (.shp) e/ou banco de dados pretendido
  2. Matriz de Vizinhança
  3. Análise exploratória Espacial (Mapas temáticos, autocorrelação espacial)
  4. Modelagem (ex: SAR, CAR ou GWR)

7- Discutir os resultados em ambas as etapas.

8- Entregar um relatório escrito em PDF no formato de artigo com introdução, objetivos, metodologia, resultados e sua respectiva análise e uma breve discussão.

9- Não esquecer de compactar todos os scripts e dados em um unico arquivo com seu nome ou nome da dupla e data (por exemplo Wagner_Oswaldo_04Fev2020.ZIP).

10- O prazo FINAL para envio dos trabalhos é 29 de Fevereiro de 2020
